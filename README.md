# Number to Word
On-site technical whiteboard exercise with UC engineering team.
 
## Specs
- Input: integer
- Output: string
- Bounds: 0 - 99999

## Usage
Given: 14215

Expected Result: "fourteen thousand two hundred and fifteen"

Example:
fnName(14215) // "fourteen thousand two hundred and fifteen"
